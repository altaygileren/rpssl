# Rock, Paper, Scissors, Spock, Lizard
A spin on the traditional game of Rock, Paper, Scissors with two extra moves (Spock & Lizard) to add a bit more of an edge to the game to make it a bit more unpredictable.

 
## Built With
* HTML
* CSS
* JavaScript
